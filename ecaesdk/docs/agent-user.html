API接口定义

用户相关(user)

指定用户是否激活(user_is_valid)

  • 接口名称: user_is_valid
  • 接口说明: 指定用户是否激活

参数名称 数据类型 是否必填 默认 参数说明
$user_id String 是 无 用户名(大于3小于20的字母数字的串)
  • 语法 
      /**
   * 指定用户是否激活
   *  
   * @param string $user_id
   * @return boolean
   **/
  user_is_valid($user_id)

  • 例子: 

      <?php 
  var_dump(ecae_agent_api()->user_is_valid("demo"));

  • 返回结果 

      //
